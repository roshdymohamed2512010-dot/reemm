Cyber Security

XSS

Cross Site Scripting هي ثغرة تسمح بحقن أكواد JavaScript داخل صفحات الويب إذا لم يتم التحقق من مدخلات المستخدم.

SQL Injection

تحدث عند إرسال استعلامات ضارة إلى قاعدة البيانات. يمكن تجنبها باستخدام Prepared Statements.

CSRF

هجوم يعتمد على استغلال جلسة المستخدم. الحل هو استخدام CSRF Tokens والتحقق من الطلبات.

Password Security

استخدم كلمات مرور قوية وقم بتشفيرها باستخدام خوارزميات حديثة مثل bcrypt أو Argon2.
